# 科研图像图注编辑器

基于浏览器的生物科研图像标注工具。上传图像 → 调整 → 加边框 → 添加图注和形状 → 导出高清图像，全流程在浏览器内完成，无需安装任何软件。

## 快速开始

```bash
cd scientific-annotator
python run.py
```

浏览器自动打开，Fabric.js 和 Inter 字体从 CDN 自动加载，无需额外安装依赖。

> **注意**：必须通过 HTTP 服务器访问，因为使用了 ES Modules。直接双击 `index.html`（`file://` 协议）无法运行。

## 功能概览

### 1. 上传图像

- 点击工具栏 **"上传"** 按钮选择文件，或直接拖放图像到画布区域
- 支持格式：PNG、JPEG、TIFF、BMP、SVG
- 图像自动缩放以适配画布

### 2. 图像编辑

左侧面板 **"图像调整"**：

| 功能 | 说明 | 范围 |
|------|------|------|
| 亮度 | 整体提亮/压暗 | -100 ~ +100 |
| 曝光 | Gamma 曲线调整 | -100 ~ +100 |
| 对比度 | 增强/减弱对比 | -100 ~ +100 |
| 高光 | 仅调整亮部区域 | -100 ~ +100 |
| 阴影 | 仅调整暗部区域 | -100 ~ +100 |
| 旋转 | 旋转图像 | -180° ~ +180° |

- **裁剪**：点击左侧工具 "裁剪"，框选出保留区域，确认即可
- **重置调整**：一键恢复所有调整参数

### 3. 边框

左侧面板 **"边框"**：

- 上/下/左/右四边可**独立设置宽度**（0–500px）
- 支持**颜色选择器**自定义边框颜色
- 边框始终位于图像外侧，不影响图像内容

### 4. 图注（文字标注）

1. 点击左侧工具 **"文字"** 进入文字模式
2. 点击画布上的任意位置添加文字框，或拖拽绘制以确定文本框宽度
3. 双击文字框编辑内容
4. 选中文字框后，右侧属性面板可设置：

| 属性 | 选项 |
|------|------|
| 字体 | Arial、Times New Roman、Helvetica、Courier New、Georgia、Verdana、宋体、黑体、楷体 |
| 字号 | 8–500pt |
| 样式 | 加粗 / 斜体 / 下划线 |
| 颜色 | 颜色选择器 |

- **上标/下标**：选中文字后切换上标(X²)或下标(X₂)模式，新输入字符自动应用样式
- **文本内容同步**：属性面板中的文本域与画布文字双向同步

### 5. 形状标注

左侧工具面板提供五种形状工具：

| 工具 | 说明 | 创建方式 |
|------|------|----------|
| 花括号 | Bezier 曲线绘制的花括号 `{` | 单击或拖拽 |
| 椭圆 | 可调整长短轴的椭圆 | 单击或拖拽 |
| 矩形 | 矩形框 | 单击或拖拽 |
| 直线 | 自由线段，旋转以中心点为轴心 | 拖拽绘制 |
| 箭头 | 线段 + 实心三角形箭头，旋转以中心点为轴心 | 拖拽绘制 |

选中形状后，右侧属性面板可设置：

| 属性 | 说明 |
|------|------|
| 描边颜色 | 形状边框颜色（箭头同步更新箭头填充色） |
| 描边宽度 | 1–20px |
| 填充 | 开关 + 填充颜色（箭头始终开启填充） |
| 位置 | X/Y 坐标微调 |

- **吸附对齐**：拖拽对象靠近其他对象边缘时，8px 内自动吸附并对齐，显示辅助参考线
- **旋转吸附**：旋转对象靠近 0°/90°/180°/270° 时自动吸附

### 6. 导出

1. 点击工具栏 **"导出"** 按钮
2. 在弹出的对话框中设置：
   - **分辨率倍率**：1x / 2x / 3x / 4x（用于学术出版的高分辨率需求）
   - **格式**：PNG（无损）或 JPEG（可调质量 10%–100%）
3. 点击 **"导出下载"**，浏览器自动下载

导出图像包含：原图 + 所有调整效果 + 边框 + 文字标注 + 形状标注（含直线和箭头）。

### 7. 撤销 / 重做

- 工具栏按钮："撤销" / "重做"
- 快捷键：`Ctrl+Z`（撤销）、`Ctrl+Y`（重做）
- 最多保存 50 步历史
- 直线和箭头完整支持撤销/重做

### 8. 其他快捷键

| 快捷键 | 操作 |
|--------|------|
| `Delete` / `Backspace` | 删除选中的对象 |
| `Escape` | 取消选择 / 退出裁剪模式 |
| `Space` + 鼠标拖拽 | 平移画布视图 |

### 9. 工具栏吸附开关

工具栏 **"吸附"** 按钮可开关对齐吸附功能（默认开启）。开启时拖拽对象会自动吸附到附近对象的边缘和中心线。

## 浏览器兼容性

支持所有现代浏览器：Chrome、Edge、Firefox、Safari。

## 设计主题

采用暖调奶油色画布 + 珊瑚色主色调的编辑风格设计：

- **字体**：正文 Inter（人文主义无衬线体）+ 等宽 JetBrains Mono
- **色彩**：暖调奶油色基底（#faf9f5）+ 珊瑚色强调（#cc785c）+ 暖黑墨色文字（#141413）
- **圆角**：4px–16px 层级体系，按钮和卡片采用柔和圆角
- **间距**：4px 基准网格，面板和控件采用充裕内边距

## 技术栈

- HTML5 Canvas + Fabric.js 5.3.1
- 原生 JavaScript（ES Modules）
- CSS3 自定义属性（浅色暖调主题）
- Google Fonts（Inter + JetBrains Mono）
